📋 QUICK START (2 MINUTES)
─────────────────────────────────────────────────────────────────────────

Step 1: Navigate to Project
    cd src/AutoPartShop.Web

Step 2: Install Dependencies
    npm install

Step 3: Build Tailwind CSS
    npm run tailwind:build

Step 4: Start Development (Open 2 terminals)

    TERMINAL 1 - Watch CSS:
    npm run dev

    TERMINAL 2 - Run Application:
    dotnet run

Step 5: Open Browser
    https://localhost:7020

🎨 DESIGN SYSTEM
─────────────────────────────────────────────────────────────────────────

Primary Color (Sky Blue):
  - Button color: #0284c7
  - Hover state: #0369a1
  
Dark Color (Professional Gray):
  - Sidebar: #0f172a
  - Text: #1e293b or lighter
  - Borders: #cbd5e1

Status Colors:
  - Success (Green): Completed, In Stock
  - Warning (Orange): In Progress, Low Stock  
  - Danger (Red): Failed, Out of Stock
  - Info (Blue): Pending, Primary

❓ TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────

Styles not showing:
  → npm run tailwind:build
  → Clear browser cache (Ctrl+Shift+Delete)

npm not found:
  → Install Node.js from nodejs.org

Build errors:
  → dotnet clean
  → dotnet build

Changes not appearing:
  → Restart "npm run dev"
  → Restart "dotnet run"
